# choi-draw

🔗 **デモはこちら: [https://inajob.github.io/choi-draw/](https://inajob.github.io/choi-draw/)**

`choi-draw` は、画像に文字や矢印を書き込める、Windows 95風（LowResスタイル）の軽量なWebアプリケーションです。
HTML5 Canvas、CSS、バニラJavaScriptで構築されており、ビルドプロセスや外部ライブラリへの依存なしに、ブラウザで直接動作します。

## 主な機能

- **レトロな Windows 95 風のUI**: ウィンドウ枠、タイトルバー、立体的なボタン、専用のカラーパレット、最下部のステータスバーなど、クラシックなOSのデザインを再現しています。
- **マルチデバイス対応**: PCのマウス操作とスマートフォンのタッチ操作の両方を統合したポインターイベント (Pointer Events) を使用しています。
- **解像度に応じた自動スケーリング**: 読み込んだ画像の解像度に合わせて、描画する矢印や文字の大きさを自動的にスケール調整します。これにより、低解像度の画像でも高解像度の画像でも、意図通りの比率で書き込みが行えます。
- **クラシックな16色プリセットパレット**: MSペイント風の16色パレットと、現在選択中の色を示すプレビューボックスを搭載しています。
- **Undo（元に戻す）機能**: 描画した矢印やテキストを1ステップずつ元に戻すことができます。
- **モバイルフレンドリー**: スマートフォンにフィットするレスポンシブなビューポート設定、描画中の意図しないズーム（ピンチイン・アウト）防止、および横スクロールを発生させない幅自動調整を適用しています。
- **簡単な導入**: `index.html` ファイルをブラウザで開くだけで、インストール不要で動作します。

## 使用方法

1. **アプリを開く**: ブラウザで [index.html](index.html) を開きます。
2. **画像を選択する**: 画面最上部にある「画像選択」のファイル選択ボタンから、編集したい画像を選びます。
3. **色とツールの選択**:
   - 16色プリセットパレットから描画したい色をクリックします。
   - **Text** ボタン（文字ツール）または **Arrow** ボタン（矢印ツール）をクリックして切り替えます。
4. **描画する**:
   - **Arrow モード**: キャンバス上をドラッグして矢印を描画します。
   - **Text モード**: ボタンの下に出現するテキスト入力欄に描画したい文字を入力します。その後、キャンバス上をタップ/クリック・ドラッグして位置を調整し、指やマウスを離すことで確定します。
5. **保存する**: **Save** ボタンをクリックするとプレビューが表示されます。**Open in New Tab** ボタンをクリックして別タブで統合された画像を開き、端末に保存してください。
6. **元に戻す**: 失敗した場合は **Undo** ボタンをクリックして1つ前の状態に戻せます。
